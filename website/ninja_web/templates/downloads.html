For Ubuntu Users:

You can add the NINJA-IDE PPA and install it from there (you will get automatic updates!)

sudo apt-add-repository ppa:ninja-ide-developers/daily (Daily updates)
or
sudo apt-add-repository ppa:ninja-ide-developers/ninja-ide (Stable Releases)

sudo apt-get update
sudo apt-get install ninja-ide
